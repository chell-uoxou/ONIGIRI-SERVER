Minecraft peにサーバーを追加する



Add server to Minecraft PE





  1. Minecraft PEを起動
  2. Friendタブをタップ
  3. フレンドを追加の隣にあるボタンをタップ
  4. 任意のサーバー名とIP、Portを入力※IP、Portは後記
  5. 追加ボタンをタップ


サーバーに接続する



Connect to ONIGIRI SERVER





サーバー運用中に追加されたサーバーリンクをタップすると

サーバーに入室することができます。



ログイン認証のための登録



Register for login authentication





当サーバーではプレーヤーネームの重複を防ぐために、ログイン認証を行っていただいています。

初回接続時は、チャット欄に任意のパスワードをそのまま入力してください。/register <password>などの構文は不要です。

また、接続するデバイスやIPアドレスが変更された際にパスワードの入力を要求されるため、同様に初回で設定したパスワードをチャット欄に入力してください。

注意：パスワードの要求がない場合にチャット欄に個人のパスワードを入力することで、サーバーに接続しているユーザーに知らされてしまうため、絶対に入力しないでください。




サーバーでの生活を進めるために

建築資材を採集する



Collect resources





自分の拠点を建築ワールドに築くために資源を採集しましょう！

Welcomeワールドでは前を向いて左手にあるモンスタースポナー、ほかのワールドでは「資源ワールドへワープ」と書いてある看板の上のモンスタースポナーをタップするか、コマンド"/warp shigen-spawn"を実行して資源ワールドへとテレポートしましょう！



資源ワールドは、白樺、松、樫の３つの木を採取するエリアと、鉱石、砂や砂岩を採取するエリアの５つのエリアで構成されており、自由に設置や破壊を行うことができます。



ここで、資源を採集する前に仕事につかなければなりません。

「仕事に就く」の看板をタップするか、コマンド "/job join worker"を実行してください。

仕事に就くことによって、土地を購入したり、物品を購入するための資金を得ることができます。
