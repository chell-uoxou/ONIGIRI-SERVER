Features of ONIGIRI Server



おにぎりサーバーとして初めて運営を行ってから約2年が経過。2017年度の新しくなったおにぎりサーバーが持つ特徴をご説明いたします。



Features



Anti-Griefing



もう荒らさせない！



荒らし対策は万全です。

建築用のワールドにおいては、土地を購入しないと編集が不可能となっています。また、定期的なバックアップにより、もしもワールドデータの破損が発生したり、大規模な荒らしに見舞われてしまった場合に復旧がすぐに行えるため、快適かつ安全な生活をお楽しみいただけます。



Various worlds



複数ワールド構成



おにぎりサーバーは鯖民様のニーズに合わせて大きく３つのワールドで構成されています。まずは地形やバイオームが存在する建築ワールド１、平面で建築がしやすい建築ワールド２、資源採集用の資源ワールド、の３つに分かれており、どの地点にいてもコマンド一つでテレポートが可能です。



Complete service



困ったときは助けます



私たちサーバー管理チームは常に鯖主様の要望に応えられるようにサービスの改善に尽力しております。まだまだ問題点やバグがあるため、それらを発見したときや、迷惑行為の被害などがあったときはすぐに当サーバーのスタッフにお声掛けください。

サーバーにスタッフがいない場合は、お手数ですが、当サイトの問い合わせフォーム、鯖主のTwitter、公式Lobiなどにてご報告お願いします。



Server is fast enough



そこそこ高速



それなりに性能のあるPCでそれなりに速い回線を使っているので、そこそこ高速です。４０人までで十分安定した運用ができた実績があります。大人数でのイベントなどにももってこいです。



Economic system


安定した経済システム


当サーバーでは経済システムを導入しています（Economy API）。

土地の値段が基本的に変動しないため、物品の相場があまり変化しない安定した経済システムが成り立っています。

鯖民様同士の貿易や商売、雇用主が存在する職業や、従業員型の運営なども可能です。


※経済システムに関する情報は、コマンドの使用方法などのページにてご確認ください。
